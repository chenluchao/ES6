2.input新类型

	q.常用的input的type?
	
		text
		password
		button
		submit/reset
		hide
		...
		
	q.h5引入的新type类型有哪些?
		颜色 color
			使用:(1)加上直接显示,通过value设置初始值
				(2)通过onchange获取变化
				
				注意: 使用不能用颜色名,用十六进制颜色
		
		时间和日期
			month
			week
			date
			datetime
			datetime-local
			time
			
			注意1: 初始值的格式怎么记?
				不记
			注意2: 时间和日期用datetime-local
			
		数字
			number 
			实例: 购买商品 
			
		搜索
			search
			特点: 自带清除按钮,有内容显示清除
			
		url
			url 带验证,有内容如果提交,有提示
			
		email
			url 带验证,有内容如果提交,有提示
			
		datalist 实现输入提示

3.input新属性

	q.有哪些属性
		刚用的: min max step
		
		required 必需的
			特点: 加上必须输入,不输入提示
		
		multiple 多选
			特点: 加上以前单选,可以多选,shift
		
		autofocus 自动聚焦
			特点: 页面加载自动把光标放在输入框
		
		pattern 正则表达式检查内容
			特点: 根据pattern对应格式检查数据
