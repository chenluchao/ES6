1.ES6简述

	q.什么是es6?
		es6是ECMAScript 6.0的简称, 是javascript的下一点版本
	发布于2015年6月
		
	q.为什么不叫js6.0, 为啥叫es6
		很早以前, Java比较火,js的公司,语言命名为javascript
	ECMA组织制定js标准, java是有版权的,java版权最早是sun公司
	Oracle收购了sun,新标准命名为ECMAScript 6.0
	
	q.es6支持情况
		chrome现在版本几乎支持所有es6特性
		
		注意1: 有个node工具babel可以做es6转es5

2.变量
	let
		q.什么是let?
			注意1: js中变量的作用域只有全局和函数,没有局部
	
		q.let如何使用
			let num=0;
			
		q.let的特点
			让这个变量的作用域到这个区块结束为止
			let定义的变量具有块级作用域(局部作用域)
			
		q.let作用
			不想让上面的变量干扰下面的,就使用let
			常见用途: 推荐大家把所有var变为let
	
	const
		q.什么是const变量
			const是关键字,用于定义只读变量
			
		q.如何使用?
			const pi = 3.14159
			
		q.const用在什么地方?
			定义一些不变的量,pi,e,author..
			特点: 改动会报错
			
		q.const注意什么?
			(1).初始化的时候需要赋值
			(2).不要修改,修改会报错
	
	解构赋值 
		q.什么是解构赋值?
			以前变量赋值, 单个 = 单个
			现在     结构数据 = 结构数据
	
		q.一个简单例子
			var [a,b] = [10,20]
			
		q.另外一个简单例子
			var {'name':n,'age':a} = 
				{'name':'zhangan','age':'78'}
				
		q.一般什么时候用?
			(1)函数返回数据不是单个数据,是一个复杂解构
			(2)从json中取数据(难用)
	
3.函数
	函数参数默认值
		q.什么是函数参数默认值?
			有的时候, 函数有参数,有多个,其中一些设置默认值
			
		q.如何使用?
			function sum(x,y)
			function sum(x=10,y=20)
			
		q.特性
			这个参数如果给了值, 用给的值,没有给,用默认值
	
	对象中函数简写(vue中)
		q.以前对象如何加函数?
			var dict = {
				sum:function(x,y){
					return x+y
				}
			}
			
		q.可以简化
			var dict = {
				sum(x,y){
					return x+y
				}
			}
			
	箭头函数[重要] (html,vue,node)
		q.什么是箭头函数?
			是匿名函数的简写
			
		q.写一个简单的箭头函数
			
			
		q.箭头函数的好处
			(1)简化回调,少写代码
			(2)箭头函数作用域是调用处的作用域
				不写var self = this
	
	
	rest参数(不定参数)
		q.是啥玩意?
			实现了给函数传递不定个参数
			
		q.如何使用
			function func(p1,p2, ...list)
			func(10,20)
			func(10,20,'a')
			func(10,20,'a','b')

	类,对象,继承,初始化函数继承

			直接使用class
			格式
				class 类名{
					constructor(){
						this.属性名 = value
					}
					say(){
						
					}
				}
	
	extends
		q.es6如何继承一个类
			格式
				class 类名 extends 父类{
					constructor(){
						super(父类初始化参数)

					}
					say(){
						
					}
				}

		现实代理: 你不方便的事情, 交给谁中介去做,中介人就是代理
		
		js中proxy:  给原对象添加一个代理对象, 操作代理对象,监控所有set和get
		
		为什么使用: 
	
	q.如何使用
		核心: 给原对象创建一个代理对象
		
		
	q.proxy的应用
		vue3.0底层监控数据改变, 可能会使用
		
		实现属性的只读变量
		
		数据保护
